# Sportkalender iOS (Native)

Native iOS-App mit SwiftUI.

## Projekt öffnen

1. Xcode starten.
2. Projekt öffnen:
   - `/Users/nico/Documents/Development/sportkalender/ios-native/SportkalenderiOS.xcodeproj`
3. Scheme `SportkalenderiOS` wählen.
4. Auf Simulator oder iPhone starten.

## Features

- Liga- und Saisonauswahl
- Wochennavigation
- Tagesweise Spieleliste
- Ergebnisanzeige
- TV-Mapping
- Live-Daten über OpenLigaDB

## Hinweise

- Das Projekt wurde mit `xcodegen` erzeugt (`project.yml`).
- Wenn du die Projektstruktur änderst, danach neu generieren:

```bash
cd /Users/nico/Documents/Development/sportkalender/ios-native
xcodegen generate
```
